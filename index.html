Wahyumart

I developed this website as a solution for managing inventory and sales at WahyuMart in an efficient and structured manner. Using the Laravel framework, I ensured that this system is not only secure and reliable but also equipped with modern features that facilitate stock monitoring, transaction management, and sales reporting. This development aims to support smooth operations and assist management in making informed decisions based on accurate inventory and sales data.

This website includes the following features and pages :

  • Login Page: Contains a login page where users can enter a username and password.
  • Home Page: Displays an inventory table with item names, stock, prices, and actions to edit, view details, and delete items.
  • Sale Page: Contains a sales page to input transactions, including a sales table that lists item names, prices, and total prices. The sales bar is used to add transactions to the sales table, and the transaction button loads all transaction data into the transactions table.
  • Transaction Page: Contains a transactions table listing item names and a print button to print existing transaction data.
